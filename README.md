# 🌐 Personal Portfolio Website

This is my personal portfolio website designed to showcase my skills, projects, and experience as a software developer. It serves as a central platform for recruiters and collaborators to explore my work and learn more about me.

---

## 🚀 Live Demo

👉 https://portfolio-website-sigma-jade-97.vercel.app/

---

## ✨ Features

* 🧑‍💻 About Me section highlighting my background and journey
* 💼 Projects showcase with descriptions and links
* 🛠️ Skills section covering my technical stack
* 📱 Fully responsive design (mobile, tablet, desktop)
* 🎨 Clean, modern UI with smooth layout
* ⚡ Fast loading and optimized performance
* 🌙 Dark mode support

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Styling:** Tailwind CSS
* **Version Control:** Git & GitHub
* **Deployment:** Vercel

---

## 📁 Project Structure

```bash
Portfolio-website/
│
├── assets/          # Images and icons

├── main.jsx       # Main entry point
└── README.md
```

---

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/fellygor/Portfolio-website.git
cd Portfolio-website
```

Run locally:

```bash
# For static version
Open index.html in your browser

# If using a framework
npm install
npm run dev
```
